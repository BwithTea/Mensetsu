# Mocksy

## Overview
Mocksy is a web based mock interview application focused on helping users practice behavioral interview questions through recorded responses.

## Purpose
Provide a simple and realistic way to prepare for interviews by answering questions verbally and reviewing performance.

## Current Features
Reusable Button and TextBox components  
Home page for basic interaction testing  
React Router setup  

## Tech Stack
React (JavaScript, JSX)  
React Router v7  
Vite  

## Status
Early development. Current focus is core structure and reusable components. Future work includes audio recording and AI driven interview flow.
