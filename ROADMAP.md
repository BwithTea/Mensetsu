# Mocksy Road Map

## Goal
Help users practice behavioral interviews by simulating a realistic mock interview experience.

## Core Features
- Start a mock interview session
- AI asks behavioral interview questions (text only for MVP)
- User records audio responses
- Save responses locally for the session
- End session with basic AI feedback

## Implementation-Goals 
- Video recording
- Advanced scoring
- User accounts / Authentication
- Sharing or exporting interview for more professional feedback
- Voice AI interviewer "Text to speech"

## Tech (Initial)
- Frontend: React
- Backend: Node.js
- AI: API based (no model training)
- Storage: In-memory or temporary

## Success Criteria
- App runs locally
- User can complete one full mock interview
- Audio is recorded and replayable
- Feedback is generated at the end
